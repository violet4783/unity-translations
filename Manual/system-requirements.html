    XR platform system requirements

    As part of the migration to the Unity XR plugin architecture using the XR Management package, built-in platform packages will be deprecated in Unity 2019.3 and newer versions.

    These implementations are still functional and available in the Unity Package Manager, but bugs and issues might not be prioritized and the package will be removed in future releases.

    For Google Daydream/Cardboard, OpenVR, and Vuforia support options in Unity 2019.3+, see the XR plugin architecture page.
